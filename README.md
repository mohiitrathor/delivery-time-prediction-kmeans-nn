# Food Delivery Time Prediction using Machine Learning

## Overview
This project focuses on predicting food delivery time using a combination of clustering and neural networks. It simulates real-world delivery scenarios by analyzing multiple factors affecting delivery duration.

---

## Tech Stack
- Python  
- NumPy, Pandas  
- Scikit-learn  
- TensorFlow / Keras  
- Matplotlib / Seaborn  

---

## Approach

### Data Preprocessing
- Handled missing values  
- Feature encoding & normalization  
- Exploratory Data Analysis (EDA)

### Clustering
- Applied **K-Means Clustering** to group delivery patterns  
- Helped in identifying hidden structures in delivery data  

### Model Building
- Built a **Neural Network model** for predicting delivery time  
- Optimized using appropriate loss functions and optimizers  

### Evaluation
- Performance evaluated using:
  - Mean Absolute Error (MAE)  
  - Root Mean Squared Error (RMSE)  

---

## Key Features
- Real-world problem simulation  
- Combination of **unsupervised + supervised learning**  
- End-to-end ML pipeline  
- Data-driven insights
